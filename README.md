# python-functions-intern

Short overview
This repo contains exercises covering Python functions: definition, parameters, *args/**kwargs,
lambdas, higher-order functions, recursion, decorators, and edge cases.

How to run tests
1. Create a virtual environment:
   python -m venv .venv
   source .venv/bin/activate   # on Windows: .venv\Scripts\activate
2. Install requirements:
   pip install -r requirements.txt
3. Run tests:
   pytest -q

Structure:
- src/: source files
- tests/: pytest unit tests
- solutions/: (optional) reference solutions after submission
